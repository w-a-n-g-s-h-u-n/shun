# 微服务快速开发脚手架
    基于Spring Cloud Hoxton.SR4的微服务开发脚手架
## 仓库地址
``` xml
<repositories>
    <repository>
        <id>shun</id>
        <name>shun</name>
        <url>https://raw.githubusercontent.com/w-a-n-g-s-h-u-n/shun/mvn-repo</url>
        <snapshots>
            <enabled>false</enabled>
        </snapshots>
    </repository>
</repositories>
```
